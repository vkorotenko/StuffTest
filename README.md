# StuffTest
## Описание

Использовал Visual Studio 2022 шаблон проекта Web API Controller.
В качестве БД использована sqllite.
Ограничения на логику накладывал как с помощью Fluent API так 
и с помощью DataAnnotation. 


## Запуск 
Просто скачать и запустить в отладочном режиме. 
http://localhost:5241/swagger

Приложение автоматически создает 3 пользователей с разными ролями. 
При ошибке выдается статус 400 и модель ErrorModel
